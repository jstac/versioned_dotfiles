# versioned_dotfiles


## Install steps

* install rofi, latex, anaconda, julia

* add the following to `.profile`

```

# Permanently map CAPS to CTRL
/usr/bin/setxkbmap -option "ctrl:nocaps"

# Start rofi
rofi -key-run F12 &

```
